# RNASeqGSEA2026

RNA-Seq Gene Set Enrichment Analysis — 2026

## Project Overview

This project performs RNA-Seq differential expression analysis followed by Gene Set Enrichment Analysis (GSEA).

## Repository Structure

```
RNASeqGSEA2026/
├── data/        # Raw and processed data files
├── scripts/     # Analysis scripts (R, Python, bash)
└── README.md    # Project documentation
```

## Getting Started

1. Place raw FASTQ or count matrix files in `data/`
2. Run analysis scripts from `scripts/`

## Requirements

- R (≥ 4.0) with DESeq2, clusterProfiler, ggplot2
- Python (≥ 3.8) with pandas, numpy (optional)
